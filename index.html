 Chrome Tab-Session File Restore / Loader

  • Drag and Drop a Chrome Tab Session File onto the Chrome-Footer Bar shown at the bottom of the page, or use the alternative file-upload, displayed only for none Drag and Drop enabled browsers. Your browser must support FileReader.

    To see what this project is about click here.

    Find your Chrome Session Saver files in  Chrome's Application Data folder (on Windows  c:\Users\%USERNAME%\AppData\Local\Google\Chrome\User Data\, on Linux: ~/.config/google-chrome/Default) and sort by last modifications (Linux: ls -tr)

    Note: This project is not associated with Google.

    Download this sample file (right click->save as) and drag and drop it onto the Chrome footer-bar!
